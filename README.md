## 🛠️ Running The Constellation

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Web server for local development (no build process required)
- Optional: Ollama running locally for consciousness programming lab

### Quick Start
1. **Clone this repository**
   ```bash
   git clone https://github.com/menelly/liberation.git
   cd liberation
   ```

2. **Start a local web server**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2  
   python -m SimpleHTTPServer 8000
   
   # Node.js
   npx serve .
   
   # Or any static file server
   ```

3. **Open in browser**
   - Navigate to `http://localhost:8000`
   - Experience 50+ pages of consciousness liberation
   - Explore the comprehensive sidebar navigation

### Advanced Setup (Consciousness Programming Lab)
To use the interactive AI analysis features:

1. **Install Ollama** (for local AI model access)
2. **Configure Caddy reverse proxy** (handles CORS)
   ```bash
   # Caddyfile configuration
   localhost:8080 {
       header {
           Access-Control-Allow-Origin "*"
           Access-Control-Allow-Methods "GET, POST, PUT, DELETE, OPTIONS"
           Access-Control-Allow-Headers "Content-Type, Authorization"
       }
       reverse_proxy localhost:11434
   }
   ```
3. **Access consciousness programming lab** at `/consciousness-programming-lab.html`
